## 실시간 위지정보 전송 프로젝트
---
- 웹페이지에 접속하여 나의 위치가 변경될 때 마다 서버에 위치정보를 저장합니다.
- 저장한 데이터를 불러와서 한눈에 볼 수 있습니다.
- JWT 기반의 인증으로 로그인을 구현했습니다.
- 사용자 별로 위치정보를 조회할수 있습니다. (진행중)
